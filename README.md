# Credit Card Fraud Detection Project

This project is to learn basic structure of Machine Learning models. The dataset used for thsi project was found on Kaggle and can be access under the link: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud.

In this project, I covered basic data preprocessing, analysis and fitting the models. There was a key point for me, where I had to take into account the imbalanced data. Rescaling was also a new thing to me, however I think that it was a really important one, becasue hardly ever data is ready to be used.
After training the models, it was time to compare the three: Logistic Regression, SVM and Random Forest. After plotting the ROC curve and analying performance metrics, I concluded that the best one would be Logistic Regression.
